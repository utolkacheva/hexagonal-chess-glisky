Шестиугольные шахматы Глинского. Задание #14
Java-приложение для игры в шестиугольные шахматы Глинского с поддержкой двух режимов игры и графическим интерфейсом на JavaFX.

Шестиугольные шахматы Глинского — вариация классических шахмат. В отличие от традиционных шахмат с квадратной доской 8×8, эта версия использует шестиугольную доску с 91 полем.

Аргументы командной строки
Игра поддерживает два режима запуска через аргументы командной строки:

"Хочу играть!" - интерактивный режим

Опционально: WHITE или BLACK (цвет игрока)
Запуск осуществляется с помощью команды mvn javafx:run "-Dexec.args=Хочу играть! WHITE"
`Я наблюдатель` - режим автоигры

Запуск осуществляется с помощью команды mvn exec:java "-Dexec.mainClass=ru.vsu.cs.tolkacheva_u_a.task2.Main" "-Dexec.args='Я наблюдатель'"
Движение фигур
1. Пешка (Pawn)
Ход вперед: На одно поле по направлению к противнику
Начальный ход: Может пойти на два поля вперед с исходной позиции
Взятие: По диагонали (специальные направления для каждого цвета)
Превращение: В ферзя при достижении последней горизонтали
2. Ладья (Rook)
Ходит по прямым линиям в 6 направлениях
Любое количество полей, пока путь свободен
3. Конь (Knight)
Ходит "буквой Г" (12 возможных ходов)
Единственная фигура, которая может перепрыгивать через другие
4. Слон (Bishop)
Ходит по диагоналям своего цвета (6 направлений)
Остается на полях того же цвета, на котором начал
5. Ферзь (Queen)
Объединяет возможности ладьи и слона
Самый мощная фигура на доске
6. Король (King)
Ходит на одно поле в любом из 6 направлений
Не может ходить под шах
